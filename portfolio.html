Projects Completed

Built a Convolutional Neural Network

I implemented a CNN in Python using PyTorch and SciKit-Learn to classify clothing images from the fashion-mnist dataset. The model's architecture featured an Adam optimizer with Cross Entropy Loss and was trained over 30 epochs, comprising 84,058 parameters. The structure included two Convolutional layers, two fully connected layers with dropout, and concluded with the output layer. Ultimately, the model achieved an impressive 89.6% accuracy during testing.
