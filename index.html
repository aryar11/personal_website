Seq-2-Seq Translation Model

Deep Learning Clas Project

In a MS level deep learning course, our final project was to build a seq2seq model that translates between two artificial languages in order to test our ability apply what we learned about embeddedings, transformers, and hyperparameter tuning. I designed an encoder-decoder architecture that used transformer components like multi-head self-attention and residual connections. For preprocessing, I tokenized the input text, normalized case, and padded sequences to ensure consistent input lengths, and also built custom vocabularies for both languages. After carefully tuning hyperparameters such as learning rate and dropout, my model achieved 98% test accuracy.

Real vs AI CNN

Machine Learning Project

In this personal project, I wanted to challenge myself to see if I can build a model that would be able to determine whether an image is AI generated.
For this model, I used PyTorch and Tensorflow to help me with this. I was able to get my CNN to classify real vs AI images with an accuracy 80% and F1 score of 0.66 for DALL-E 2 images. The model starts with a series of convolutional layers using ReLU activations to extract hierarchical features from the input images. Max pooling layers follow these convolutions to reduce the spatial dimensions and retain the most important information. To help the model generalize, I incorporated batch normalization and dropout regularization. The extracted features are then flattened and passed through fully connected layers, with a softmax activation at the end to produce class probabilities. For preprocessing, I applied standard techniques such as image normalization and data augmentation, which helped improve the modelâ€™s robustness and performance.
